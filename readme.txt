Proyecto RCIS

Web para el grupo de excelencia de Universidades españolas que reunían a los grupos de investigación: en la web se detallan las herramientas, información, enlaces, universiades colaboradoras, etc... tomando como fuente de datos ficheros de Google Drive compartidos entre todos los profesores y doctores de las distintas Universidades.

Contiene una parte privada para fomentar la compartición de documentación e ideas entre los distintos grupos de investigación.
